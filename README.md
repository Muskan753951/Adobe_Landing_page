# 📱 PopX Mobile UI - React Assignment

## 🚀 Project Overview

This project is a **pixel-perfect mobile UI implementation** based on the provided Adobe XD design.
It includes multiple screens with smooth navigation and a centered mobile layout on the web.

---

## ✨ Features

* 📱 Mobile-first UI (centered on webpage)
* 🎨 Pixel-perfect design (based on Adobe XD)
* 🔄 Seamless page navigation
* 🔐 Basic authentication using localStorage
* ✅ Form validation (Login & Signup)
* 👤 Dynamic user data on Account page

---

## 📄 Pages Implemented

1. **Home Page**

   * Welcome screen
   * Navigation to Login & Signup

2. **Login Page**

   * Email & Password input
   * Validation + user authentication

3. **Signup Page**

   * User registration form
   * Stores user data in localStorage

4. **Account Settings Page**

   * Displays logged-in user info
   * Protected route (cannot access without login)

---

## 🛠️ Tech Stack

* ⚛️ React JS (Vite)
* 🎨 Tailwind CSS
* 🔀 React Router DOM
* 💾 LocalStorage (for authentication)
